# BLS Ingredient Mapping Guide

## Overview

This document provides mapping recommendations for the top 50 unmatched ingredients from the Cookidoo recipe database to the BLS (Bundeslebensmittelschlüssel - German Food Code) database entries. The mappings are based on frequency analysis of 876 recipes.

## Top Unmatched Ingredients by Frequency

## Implementation Notes

### Handling Ambiguous Cases

1. **Speisestärke (Cornstarch vs Potato Starch)**
   - Default: Kartoffelstärke (BLS 14041)
   - Consider context clues (Asian recipes may indicate corn starch)
   - Alternative: Maisstärke if explicitly indicated

2. **Generic "Curry"**
   - Mapped to: Currypulver (BLS 1021)
   - Note composition varies
   - Consider Indian vs Thai curry type if context available

3. **Fresh vs Dried Herbs**
   - Mappings assume dried unless specified
   - Look for keywords: "frisch", "getrocknet", "Kraut"
   - Adjust BLS entry accordingly

4. **Pasta Types**
   - Generic "Nudeln" → Spaghetti (20078)
   - Specific shapes have same BLS code usually
   - Linguine/Bandnudeln → flat ribbon variant (20077)

### Quality Assurance

When implementing mappings:
- Verify ingredient form (fresh vs dried vs frozen)
- Check recipe context for ingredient-specific variants
- Cross-reference with original Cookidoo recipe instructions
- Consider regional food preferences (German cuisine focus)
